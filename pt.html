Construo sistemas de dados e IA que melhoram operações e decisões em saúde

Desenho e construo sistemas ponta a ponta de dados e machine learning, da infraestrutura de dados até aplicações prontas para produção, com foco em operações em saúde e suporte a decisão clínica.

Meu trabalho é transformar problemas operacionais complexos em sistemas de dados práticos que possam ser usados na tomada de decisão do mundo real.

O que eu construo

  • Modelos preditivos para tomada de decisão operacional
  • Pipelines de dados e infraestrutura analítica
  • Sistemas de conhecimento com IA para acesso a protocolos e processos
  • Sistemas de BI e suporte a decisão para equipes de saúde

Projetos selecionados

  • Agente conversacional para consulta de protocolos operacionais
    IA para saúde RAG Sistemas de conhecimento

    Desenvolvi um assistente conversacional para ajudar equipes operacionais em saúde a acessar rapidamente protocolos e procedimentos internos, reduzindo tempo de busca e aumentando a consistência no atendimento.

    O projeto envolveu a construção de um sistema de conhecimento baseado em RAG estruturado para o contexto de centrais de atendimento, em que acesso rápido e confiável a informação impacta diretamente a qualidade operacional.

    O trabalho incluiu pipelines de ingestão documental, extração da estrutura de protocolos, definição de estratégias de chunking alinhadas aos fluxos operacionais, desenho de metadados e indexação semântica de grandes coleções de documentação institucional.

    Houve atenção especial a qualidade de recuperação, relevância contextual e rastreabilidade das respostas, permitindo vincular cada resposta a trechos específicos dos protocolos utilizados no atendimento.

    O resultado foi um fluxo de conhecimento orientado a produção, pensado para reduzir atrito operacional e tornar decisões baseadas em protocolo mais consistentes entre equipes de atendimento.

  • Machine learning para predição de faltas em consultas
    Machine learning Operações em saúde Analytics preditivo

    Construí modelos preditivos para identificar consultas com alto risco de não comparecimento, permitindo ações operacionais proativas para reduzir faltas e melhorar a utilização de recursos.

    O projeto começou com uma fase extensa de discovery voltada a compreensão do contexto operacional de agendamentos e dos impactos das faltas no funcionamento dos serviços de saúde.

    Essa etapa incluiu análise exploratória de históricos de agendamento, identificação de padrões temporais e comportamentais, investigação de fluxos operacionais e discussões com stakeholders para entender como previsões poderiam apoiar decisões reais.

    A partir desse entendimento foram desenvolvidas estratégias de engenharia de atributos capazes de capturar padrões relacionados ao histórico do paciente, características do agendamento e fatores temporais.

    Diferentes abordagens de machine learning foram avaliadas considerando desempenho preditivo, interpretabilidade e aplicabilidade prática dos modelos no contexto operacional.

    O projeto foi desenhado para apoiar o planejamento operacional com sinais de risco antecipados, ajudando equipes a agir antes que ausências se convertessem em capacidade desperdiçada.

  • Machine learning para suporte a prescrição de diálise
    Machine learning Suporte a decisão clínica Pesquisa

    Projeto de pesquisa de graduação voltado ao desenvolvimento de modelos de machine learning para apoio a decisão em prescrição de diálise com base em variáveis clínicas.

    O trabalho explorou o uso de dados clínicos estruturados para apoiar decisões terapêuticas em contextos de tratamento dialítico.

    O projeto combinou análise exploratória, engenharia de atributos, desenvolvimento de modelos e avaliação de desempenho preditivo em um cenário clinicamente relevante.
