Sistemas de dados para saúde · machine learning · IA baseada em conhecimento

Trabalho com sistemas de dados e aplicações de machine learning voltadas a operações em saúde e suporte à decisão clínica.

Minha atuação envolve construção de infraestrutura analítica, modelos preditivos e sistemas de conhecimento que auxiliam decisões operacionais e clínicas.

Projetos selecionados

  • Agente conversacional para consulta de protocolos operacionais
    IA para saúde RAG Sistemas de conhecimento

    Desenvolvimento de um assistente conversacional voltado ao suporte de equipes operacionais na consulta a protocolos institucionais e procedimentos internos.

    O projeto envolveu a construção de um sistema de conhecimento baseado em RAG estruturado especificamente para o contexto de centrais de atendimento em saúde.

    O trabalho incluiu pipelines de ingestão documental, extração da estrutura de protocolos, definição de estratégias de chunking alinhadas aos fluxos operacionais, desenho de metadados e indexação semântica de grandes coleções de documentação institucional.

    Houve atenção especial à qualidade de recuperação de informação, relevância contextual e rastreabilidade das respostas, permitindo que cada resposta pudesse ser vinculada a trechos específicos dos protocolos utilizados no atendimento.

    O objetivo do sistema foi reduzir o tempo de busca por procedimentos operacionais e melhorar a consistência na interpretação de protocolos durante interações de atendimento em saúde.

  • Machine learning para predição de faltas em consultas
    Machine learning Operações em saúde Analytics preditivo

    Desenvolvimento de modelos preditivos voltados à identificação de consultas com maior probabilidade de não comparecimento de pacientes.

    O projeto começou com uma fase extensa de discovery voltada à compreensão do contexto operacional de agendamentos e dos impactos das faltas em consultas no funcionamento dos serviços de saúde.

    Essa etapa incluiu análise exploratória de históricos de agendamento, identificação de padrões temporais e comportamentais, investigação de fluxos operacionais e discussões com stakeholders para entender como previsões poderiam apoiar decisões operacionais.

    A partir desse entendimento foram desenvolvidas estratégias de engenharia de atributos capazes de capturar padrões relacionados ao histórico do paciente, características do agendamento e fatores temporais.

    Diferentes abordagens de machine learning foram avaliadas considerando desempenho preditivo, interpretabilidade e aplicabilidade prática dos modelos no contexto operacional.

  • Machine learning para suporte à prescrição de diálise
    Machine learning Suporte à decisão clínica Pesquisa

    Projeto de pesquisa de graduação voltado ao desenvolvimento de modelos de machine learning para apoio à decisão em prescrição de diálise com base em variáveis clínicas.

    O trabalho explorou o uso de dados clínicos estruturados para apoiar decisões terapêuticas em contextos de tratamento dialítico.
